# 📚 Student Course Enrollment System

## 📌 Project Overview

This project is a simple Java-based system to manage student enrollments in courses.
It demonstrates the use of Java Collections Framework and avoids duplicate enrollments.

---

## 🚀 Features

* Add student details (ID, Name, Marks)
* Enroll students into courses
* Prevent duplicate enrollments
* Store data using ArrayList / HashSet
* Traverse data using Iterator

---

## 🛠️ Technologies Used

* Java
* Java Collections Framework
* OOP Concepts

---

## 📂 Project Structure

```
com.student.model
 └── Student.java

com.student.service
 ├── ArrayListService.java
 ├── DuplicateService.java
 ├── IteratorService.java

com.student.main
 └── Main.java
```

---

## ▶️ How to Run

1. Open project in any IDE (Eclipse / IntelliJ)
2. Make sure package structure is correct
3. Compile all classes
4. Run `Main.java`
